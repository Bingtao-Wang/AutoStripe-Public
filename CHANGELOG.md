# Changelog

All notable changes to AutoStripe will be documented in this file.

## [1.0.0] - 2026-03-13

### Added
- Complete system implementation with full source code
- Three-modal perception switching (GT, VLLiNet, LUNA-Net)
- Two-level offset path planning algorithm
- Pure Pursuit lateral controller with adaptive look-ahead
- Auto-spray state machine for line marking
- Automated experiment runner for multi-weather validation
- Manual control interface (v5 and v6)
- Evaluation scripts (metrics, visualization)
- Technical documentation and design documents
- MIT License

### Features
- CARLA 0.9.13 integration
- ROS Noetic support
- Multi-weather simulation (5 conditions)
- Real-time RViz visualization
- Comprehensive logging and metrics
